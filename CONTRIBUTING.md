# Contributing

Thanks for considering contributing!

## Development

1. Fork & clone, then install deps:
   ```bash
   npm ci
   ```
2. Build:
   ```bash
   npm run build
   ```
3. Add tests or examples if you change behavior. Keep PRs focused.

## Reporting issues

Please include:
- What you tried and expected vs. got.
- Steps/code to reproduce.
- Node/OS versions and package version.

## Pull requests

- Keep diffs minimal and clear.
- Update README/docs/examples when APIs change.
- Ensure `npm run build` passes.
